# Factory Performance Analysis

## Tools Used
- Excel
- Tableau

## Files
- factory_data.xlsx → Raw & processed data
- dashboard.twbx → Tableau dashboard
- dashboard.png → Dashboard preview

## What I Did
- Cleaned and structured raw telemetry data
- Created calculated metrics (downtime analysis)
- Built interactive Tableau dashboard

## Key Insights
- Factory Shenzhen has the highest downtime
- Laser-based devices contribute most failures
- Downtime is concentrated, not evenly distributed

## Recommendations
- Focus maintenance on high-failure device types
- Optimize operations in high-downtime factories
